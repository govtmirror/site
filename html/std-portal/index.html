     State level STD disease intervention specialists (DIS) would benefit greatly from access to select websites on the Internet to reach priority populations. As these websites are often sexually explicit in nature, DIS are often blocked from obtaining access. This website access would enable DIS novel methods of communicating with at-risk individuals and, for example, encourage testing for HIV and other STDs.

     This project provides a novel technology solution, leveraging CDC's Informatics R&D Lab, which will allow state health departments secure access to websites (both non- and sexually explicit) to adequately carry out critical prevention activities. This pilot will use cloud computing technology to dynamically create, allocate, and manage computing resources as needed to create a pilot environment that each DIS can use for website access.




This portal is a resource provided by the CDC Informatics R&D Lab in collaboration with CDC's Division of STD Prevention. This portal will continue to evolve in response to the requests of pilot users.
